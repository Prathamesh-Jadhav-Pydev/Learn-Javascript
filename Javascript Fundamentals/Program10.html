Loops in Javascript

  1. While Loop

    2. The while loop has the following syntax:

    while (condition)
    {
    // code
    // so-called "loop body"
    While the condition is truthy, the code from the loop body is executed.
    }

  2. Do...While loop

    3. do {
    // loop body
    } while (condition);

  3. For loop

    4. for (begin; condition; step)
    {
    // ... loop body ...
    }
